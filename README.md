# TreeBuilder

# Start project

In the root directory run:

```
npm install && npm start
```

Parcel runs locally at http://localhost:1234

# Tests

To run all tests:

```
npm run test
```

To run an individual test:

```
npm run test -t src/tests/[file-name]
```

There are unit tests for:

* node.js
* sortTree.js
* buildTree.js
